# 📊 실험 데이터 트래킹

## 수집 항목 (태스크별)
| 변수 | 설명 |
|------|------|
| participantId | 참여자 고유 ID |
| sequenceId | 시퀀스 (1-5) |
| taskOrder | 태스크 순서 (1-5) |
| taskType | FA/OB/SN/II/SE |
| labelLevel | 해당 태스크의 라벨 수준 (L0/L1/L2) |
| taskCompletionTimeSec | 완료 시간 (초) |
| clickCount | 총 클릭 수 |
| errorClickCount | 에러 클릭 수 |
| goalSuccess | 목표 달성 (0/1) |
| labelClick | 라벨 클릭 - L2 only (0/1) |
| decision | 최종 의사결정 |
| manipulationAwareness | 조작 인식도 (1-7) |
| fairness | 공정성 (1-7) |
| transparency | 투명성 (1-7) |
| trust | 신뢰 (1-7) |
| phone | 전화번호 (답례품) |
